# Time Series Forecast Visualizer — Frontend

This is the frontend presentation layer for the Time Series Forecast Visualizer, an interactive econometric dashboard built for structural break detection and AI-driven analysis.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite (for rapid HMR and optimized builds)
- **Data Visualization**: Chart.js (rendered via HTML5 Canvas for high-performance 60FPS graphing)
- **Styling**: Vanilla CSS with a Progressive Dark-Mode Aesthetic

## Prerequisites

Before running the frontend, ensure that:
1. You have **Node.js** (v18+) installed.
2. The **Node.js API Backend** is running locally (usually on port `3001`). The frontend relies on this backend for all mathematical computations (ARIMA, Holt-Winters, PELT) and Gemini LLM interactions.

## Installation & Setup

1. **Install Dependencies:**
   Navigate to the `frontend` directory and run:
   ```bash
   npm install
   ```

2. **Environment Variables:**
   Create a `.env` file in the root of the `frontend` directory if you need to point the API to a custom URL:
   ```env
   VITE_API_URL=http://localhost:3001/api/forecast
   ```
   *(If omitted, it defaults to `http://localhost:3001/api/forecast`)*

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   The application will be accessible at `http://localhost:5173`.

## Building for Production

To create an optimized production build:

```bash
npm run build
```

This will output the static assets to the `dist` folder, which can be served using any static web server (e.g., Nginx, Vercel, Render).

## Core Features

- **Dynamic Ingestion:** Upload raw CSV files containing macroeconomic time series.
- **Segmented Forecasting:** Visualize predictive algorithms reacting dynamically to structural breaks mathematically identified by the backend PELT service.
- **AI Analyst Panel:** View real-time, deterministic text summaries generated by Google Gemini explaining trend shifts and checking economic plausibility.
- **Comparison Overlay:** Compare multiple models (e.g., Moving Average vs. Holt-Winters) simultaneously on the same cartesian plane.
